# Markdown Cheatsheet

Here are a few simple formatting basics to help you write in Markdown.

## Adding Headers

Use hashtags (`#`) to create headers. The number of hashtags determines the level of the heading:

```md
# Main Title (Level 1)
## Section (Level 2)
### Subsection (Level 3)
#### Sub-subsection (Level 4)
```
